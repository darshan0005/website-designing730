Recommended Workflow

  1. Design created and finalized in Figma
  2. Specs reviewed using Figma Inspect or Zeplin
  3. Tickets created in Jira with design links
  4. Components synced with Storybook
  5. Questions resolved via Slack

Best Practices

  • Use consistent naming conventions
  • Keep designs updated and versioned
  • Document edge cases and states
  • Review builds against designs regularly
