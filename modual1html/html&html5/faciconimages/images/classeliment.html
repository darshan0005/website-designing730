Class Selector

The class selector matches elements by their class attribute.

It is denoted by a period (.) followed by the class name.

For example, to select elements with the class "highlight", you would use:

class will be repeated more than one times in a section

class is not unique

.highlight {
    background-color: yellow;
    color: black;
    font-weight: bold;
}

In this example, all elements with the class "highlight" will have a yellow background, black text, and bold font.

Classes can be applied to multiple elements, allowing for shared styles across different parts of a webpage.
