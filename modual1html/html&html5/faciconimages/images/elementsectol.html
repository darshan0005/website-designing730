Element Selector

The element selector matches elements by their tag name.

Example:

div {
background-color: lightblue;
}
p {
color: darkblue;
font-size: 18px;
margin: 10px;
padding: 5px;
font-family: 'Arial', sans-serif;
line-height: 1.6;
}

In this example, all <div> elements will have a light blue background, and all <p> elements will have dark blue text with specific styling.
