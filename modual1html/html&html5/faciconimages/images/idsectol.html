ID Selector

The ID selector matches elements by their unique ID attribute.

It is denoted by a hash symbol (#) followed by the ID name.

For example, to select an element with the ID "header", you would use: 

#header {
    background-color: lightblue;
    color: darkblue;
    font-size: 24px;
    padding: 10px;
}

In this example, the element with ID "header" will have a light blue background, dark blue text, and specific padding.

Remember, IDs should be unique within a page, so use them wisely!

Example HTML:

<div id="header">This is the header</div>
<p>This paragraph is styled by the ID selector.</p>
